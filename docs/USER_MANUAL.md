# User Manual - STI Merch System

## Getting Started

### For Administrators

1. **Login**
   - Username: `admin`
   - Password: `admin123`

2. **Main Menu Options**
   - **[1] Help**: System guide and navigation help
   - **[2] Users Reservations**: Manage student reservations
   - **[3] Stock Page**: View and manage inventory
   - **[4] Add or Remove Item**: Inventory management tools

### For Students

1. **Login**
   - Enter any username/password
   - Provide your Student ID
   - Select your course from the available options

2. **Main Menu Options**
   - **[1] Help**: How to use the system
   - **[2] Reserve a Item**: Make new reservations
   - **[3] Stock Page**: Browse available items
   - **[4] Your Reservations**: View your orders

## Detailed Workflows

### Making a Reservation (Student)
1. Login to the system
2. Go to **[3] Stock Page** to browse items
3. Note the **item code** of desired items
4. Go to **[2] Reserve a Item**
5. Enter the item code and quantity
6. Confirm your reservation
7. Wait for admin approval

### Managing Reservations (Admin)
1. Login as admin
2. Go to **[2] Users Reservations**
3. Choose **[2] View Pending Reservations**
4. Use **[3] Update Reservation Status** to approve/reject
5. Status options: PENDING → APPROVED → COMPLETED

### Adding New Items (Admin)
1. Go to **[4] Add or Remove Item**
2. Select **[1] Add New Item**
3. Enter item details:
   - Unique item code
   - Item name
   - Course (or "STI Special")
   - Size
   - Initial quantity
   - Price

## Tips and Best Practices

### For Students
- Always check stock availability before reserving
- Keep track of your reservation IDs
- Contact admin if you need to modify orders
- Cancel unwanted reservations to free up stock

### For Admins
- Regularly check pending reservations
- Monitor stock levels for popular items
- Update reservation status promptly
- Use meaningful item codes for easy tracking

## Troubleshooting

### Common Issues
1. **"Item not found"**: Double-check the item code
2. **"Insufficient stock"**: Item is out of stock or low quantity
3. **"Invalid credentials"**: Check username/password spelling
4. **Course restrictions**: Students can only view their course items

### Contact Information
For technical support or system issues, contact the IT department.

## System Limitations
- No data persistence between sessions
- Limited to predefined courses
- No payment processing integration
- Manual admin approval required for all reservations
